== Descrição ==

O Mask Elementor Form utiliza a biblioteca jQuery para inserir máscaras nos campos de texto de um formulário do Elementor Builder.

Os principais recursos do Mask Elementor Form incluem:
* Inserir máscara nos campos de texto através do placeholder destes campos (já que o Elementor Builder não possibilidade passagem de parâmetro, classe ou até mesmo um atributo personalizado para cada campo).

Para a inserção de máscaras personalizadas, utilizar:
